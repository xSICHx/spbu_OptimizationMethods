# Проект "Методы оптимизации функций"

## Описание
Этот проект представляет собой приложение на Qt, реализующее методы оптимизации различных функций. Программа позволяет:

- Выбрать размерность пространства.
- Выбрать функцию для минимизации (доступные функции зависят от размерности).
- Если функция двумерная:
  - Отобразить линии уровня.
  - Выбрать начальную точку кликом по графику.
- Определить область оптимизации (произвольный прямоугольник в выбранном пространстве).
- Выбрать критерии остановки:
  - Максимальное количество итераций.
  - Остановка по норме относительно заданного эпсилон и другие.
- Запустить один из двух методов оптимизации:
  - Случайный поиск.
  - Метод Ньютона.
- Отобразить сопроводительную информацию в отдельном окне.

## Сборка проекта
### Требования
Для сборки проекта необходимы:
- Qt 6.7.3 (MSVC2019)
- Компилятор MSVC 2019
- Git (для клонирования репозитория)

### Инструкция по сборке
1. **Клонируйте репозиторий**
   ```sh
   git clone https://github.com/xSICHx/spbu_OptimizationMethods.git
   cd spbu_OptimizationMethods/OptApp
   ```
2. **Откройте проект в Qt Creator**
   - Запустите Qt Creator.
   - Откройте файл проекта `OptApp.pro`.
   - Выберите версию Qt 6.7.3 с MSVC2019.
   - Настройте компилятор и сборку (если требуется).
3. **Соберите и запустите проект**
   - Нажмите "Собрать проект".
   - Запустите приложение.

## Использование
1. Выберите размерность пространства.
2. Выберите функцию для минимизации.
3. Настройте параметры (начальную точку, область оптимизации, критерии остановки).
4. Запустите один из методов оптимизации.
5. Анализируйте результаты, отображаемые в окне вывода.
