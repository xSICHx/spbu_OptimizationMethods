Detailed Description

Newton method of optimization.

A class that implements the Newton optimization method.

This class inherits from the OptMethod base class and provides implementations for optimizing a given function within a specified area using the Newton method. It includes methods for performing optimization with and without an initial point, as well as private helper methods for performing optimization steps and setting up transfer data.

Parameters
alphaThe step size parameter for the Newton method.
