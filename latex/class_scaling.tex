This class represent a uniform scaling transformation. It is the return type of \doxylink{class_scaling}{Scaling(\+Scalar)}, and most of the time this is the only way it is used. In particular, this class is not aimed to be used to store a scaling transformation, but rather to make easier the constructions and updates of Transform objects.

To represent an axis aligned scaling, use the Diagonal\+Matrix class.
