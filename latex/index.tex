\chapter{Optimization Methods Package}
\hypertarget{index}{}\label{index}\index{Optimization Methods Package@{Optimization Methods Package}}
The {\bfseries{Optimization Methods}} package provides a flexible framework for optimization tasks, supporting both console input and programmatic usage. It includes various optimization algorithms (e.\+g., Newton\textquotesingle{}s method, Random Search) and allows users to define functions, initial points, and stopping criteria.\hypertarget{index_autotoc_md0}{}\doxysubsection{\texorpdfstring{Features}{Features}}\label{index_autotoc_md0}

\begin{DoxyItemize}
\item {\bfseries{Console Input}}\+: Interactive console interface for selecting functions, dimensions, initial points, and optimization methods.
\item {\bfseries{Programmatic Usage}}\+: Direct integration into C++ programs for automated optimization tasks.
\item {\bfseries{Support for Multiple Dimensions}}\+: Handles optimization in 1D to 4D spaces.
\item {\bfseries{Customizable Stopping Criteria}}\+: Supports maximum iterations, function value changes, and gradient-\/based criteria.
\end{DoxyItemize}\hypertarget{index_autotoc_md1}{}\doxysubsection{\texorpdfstring{Example Usage}{Example Usage}}\label{index_autotoc_md1}
