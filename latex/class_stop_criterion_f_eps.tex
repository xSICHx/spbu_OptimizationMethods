\doxysubsection{Detailed Description}
A class that defines a stopping criterion based on the relative change in function values. 

This class inherits from \doxylink{class_stop_criterion_eps}{Stop\+Criterion\+Eps} and \doxylink{class_stop_criterion_area}{Stop\+Criterion\+Area} and combines their stopping criteria with an additional criterion based on the relative change in function values.
