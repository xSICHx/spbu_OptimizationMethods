\doxysubsection{Detailed Description}
A class that represents a stopping criterion based on the unnormalized difference between function values. 

This class inherits from \doxylink{class_stop_criterion_eps}{Stop\+Criterion\+Eps} and \doxylink{class_stop_criterion_area}{Stop\+Criterion\+Area} and provides a stopping criterion that checks if the absolute difference between the current and previous function values is greater than or equal to a specified epsilon value.
